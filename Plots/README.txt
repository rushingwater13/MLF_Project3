This folder holds the plots for each dataset. 
Each dataset has its own folder, inside of which are several plots. 
The plots named "Data#_#_Clusters.png" correspond to the k-means clusters, where the # is the number 
of the dataset and the second # is the number of clusters.
The files names "Data#_dbscan_Clusters.csv" correspond to the DBSCAN clusters where the # is the number of the dataset.
