# MLF_Project3

The Labeled_Data folder holds the .csv files of the data divided into their clusters.
The Plots folder holds all of the plots of the data, with and without the clustering.
The SSE folder holds the SSE v K plots and the values for SSE for the K-means method.

The Labeled_Data and Plots folders have a README.txt file inside with further explainations of the data and plots.
